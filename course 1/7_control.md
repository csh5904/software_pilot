# 주제 1: 제어 시스템과 피드백 제어의 기초

## 1. 제어 시스템의 기초

* **개념:** 제어 시스템은 우리가 원하는 목적에 맞게 대상 객체(로봇, 모터 등)의 상태를 조절하는 시스템을 뜻한다.
* **목표:** 외부에서 방해하는 힘(노이즈나 주변 환경의 변화)이 있더라도, 시스템이 우리가 처음에 정해준 목표 상태에 빠르고 정확하게 도달하여 안정적으로 유지되도록 만드는 것이 최종 목표이다.

## 2. 피드백 제어의 기본 개념

* **피드백:** 센서를 통해 현재 로봇의 상태를 실시간으로 측정하여 다시 제어 입력단으로 되돌려 보내는 과정이다.
* **목표값:** 우리가 로봇에게 원하는 이상적인 상태이다. (예: 로봇의 목표 속도 50cm/s, 목표 각도 90도 등)
* **오차:** 목표값과 센서가 읽어온 현재값 사이의 차이이다. 피드백 제어는 이 오차가 0이 되도록 모터의 출력을 실시간으로 깎거나 더해가는 과정이다.

## 3. 피드백 제어의 기본 구조

### 오픈 루프 제어 (열린 루프)

* 센서를 통한 피드백 과정이 없는 제어 방식이다.
* **특징:** 오직 사용자의 명령이나 미리 정해진 시간대로만 모터를 돌린다. 구조가 단순하고 비용이 적게 들지만, 주행 중 바퀴가 미끄러지거나 돌부리에 걸려 오차가 생겨도 이를 알아채고 스스로 수정할 수 없다.

### 클로즈드 루프 제어 (닫힌 루프 / 피드백 제어)

* 센서를 통해 출력 결과를 제어기로 계속 되돌려 보내는 루프(고리)가 완성된 방식이다.
* **특징:** 목표값과 센서 측정값을 실시간으로 비교하여 오차를 계산하고, 그 오차를 바탕으로 모터 출력을 계속 수정한다. 외부 방해 요소가 있어도 목표를 정확하게 찾아갈 수 있어 대부분의 로봇 제어에 필수적으로 사용된다.

# 주제 2: PID 제어의 개념과 수학적 표현

## 1. PID 제어의 개념 및 역할

PID 제어는 제어 기법 중 가장 널리 쓰이는 방식으로, 오차를 바탕으로 **비례(P), 적분(I), 미분(D)** 세 가지 항을 조합하여 제어 값을 계산하는 방식이다. 로봇이 목표 위치나 속도에 도달할 때 너무 느려지지 않게 주행을 재촉하거나, 반대로 목표를 지나쳐 앞뒤로 심하게 흔들리는 현상을 억제하는 핵심 역할을 한다.

## 2. P, I, D 각 항목의 역할과 수학적 표현

시간 $t$에서의 오차를 $e(t)$라고 하고, 각 제어 항의 고유 상수(이득)를 $K_p, K_i, K_d$라고 할 때 전체 제어 출력 $u(t)$의 제어 식은 다음과 같다.

$$u(t) = K_p e(t) + K_i \int_{0}^{t} e(\tau) d\tau + K_d \frac{de(t)}{dt}$$

### P 제어 (비례 제어: 현재의 오차 반영)

* **수학적 표현:** $K_p e(t)$
* **역할:** 현재 발생한 오차의 크기에 정확히 비례하여 제어 힘을 준다. 오차가 크면 모터를 강하게 돌리고, 오차가 작아지면 모터 힘을 부드럽게 줄인다.
* **특징:** P 상수($K_p$)를 너무 높이면 목표에 빠르게 가지만 과하게 꺾이면서 흔들림(오버슈트)이 생기고, 목표치 직전에서 힘이 모자라 완전히 도달하지 못하는 잔류 오차(정상상태 오차)가 남는다.

### I 제어 (적분 제어: 과거의 오차 누적 반영)

* **수학적 표현:** $K_i \int_{0}^{t} e(\tau) d\tau$
* **역할:** 시간이 지나도 사라지지 않고 계속 남아있는 미세한 잔류 오차들을 시간에 따라 계속 더해나가다가(누적), 이 누적된 힘으로 로봇을 밀어주어 잔류 오차를 완전히 없앤다.
* **특징:** 너무 오랫동안 쌓인 오차 때문에 목표치를 지나친 후에도 모터가 과하게 돌 수 있어 시스템이 출렁거리거나 불안정해질 위험이 있다.

### D 제어 (미분 제어: 미래의 오차 예측 반영)

* **수학적 표현:** $K_d \frac{de(t)}{dt}$
* **역할:** 오차가 변화하는 속도(기울기)를 감지한다. 로봇이 목표치에 너무 빠른 속도로 접근하면 오차가 급격히 줄어들게 되는데, 이때 미분 값이 음수가 되면서 일종의 브레이크(제동) 역할을 하여 목표를 지나치지 않게 완충해 준다.
* **특징:** 흔들림을 잡아주고 안정성을 높여주지만, 센서 데이터에 튀는 노이즈가 섞여 들어오면 미분 값이 무작위로 급증하여 제어 출력이 덜덜 떨리는 부작용이 생길 수 있다.

# 주제 3: PID 튜닝

## 1. PID 튜닝의 정의

* PID 튜닝은 로봇이 가장 안정적이고 빠르게 목표를 추종할 수 있도록 세 가지 제어 상수인 $K_p, K_i, K_d$의 최적값을 찾아내어 설정하는 과정이다.

## 2. 일반적인 튜닝 순서 (경험적 방법)

1. **P 상수($K_p$) 먼저 올리기:** $K_i$와 $K_d$를 모두 0으로 둔 상태에서, $K_p$ 값을 조금씩 올한다. 로봇이 목표치 근처까지 빠르게 가되, 목표치 주변에서 앞뒤로 가볍게 흔들리기 시작하는 지점 직전까지 설정한다. 기계적인 부품으로 거리 기반 PID 제어일 때 목표점에 달린 **스프링** 역할을 한다.
2. **D 상수($K_d$)로 흔들림 잡기:** P 제어 때문에 발생하는 오버슈트와 흔들림을 가라앉히기 위해 $K_d$ 값을 조금씩 올린다. 이를 통해 로봇이 흔들리지 않고 부드럽게 목표치에 감속하며 안착하게 만든다. 기계적인 부품으로 거리 기반 PID 제어일 때 목표점에 달린 **댐퍼** 역할을 한다.
3. **I 상수($K_i$)로 잔류 오차 제거:** 로봇이 다 멈춘 후에도 목표값과 미세한 거리 차이가 계속 남아있다면, $K_i$ 값을 아주 미량씩 주어 누적된 힘으로 그 미세한 틈을 메워 오차를 0으로 맞춘다.
4. **반복 조정:** 세 상수가 서로 영향을 주고받으므로, 로봇의 반응 속도와 안정성을 관찰하면서 정밀하게 값을 조금씩 다시 깎거나 더해가며 최종 최적점을 찾는다.
